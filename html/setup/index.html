ACEP SecComms - Getting Started

0. Prerequisites

  • Join your computer to the ACEP Headscale network.

1. Download and install the CA Certificate in your browser

Download the CA Certificate

  • ca.secdev.acep.uaf.edu.crt

Install CA in Chrome / Brave / Chromium

  1. Open the Settings page in Chromium or Brave: 
    chrome://settings/security
  2. Scroll down and click on Manage certificates.
  3. Switch to the Authorities tab.
  4. Click Import and select your Root CA certificate file: 
    ca.secdev.acep.uaf.edu.crt
  5. During the import process:
    • Check Trust this certificate for identifying websites.

Install CA in Firefox

  1. Open Firefox.
  2. Navigate to Settings:
    • Go to the menu (three horizontal bars in the top right) > Settings > Privacy & Security.
  3. Scroll to the Certificates section and click View Certificates.
  4. Switch to the Authorities tab.
  5. Click Import:
    • Select your Root CA certificate file: 
      ca.secdev.acep.uaf.edu.crt
    • Check Trust this CA to identify websites during the import process.
  6. Restart Firefox.

2. Add local host "DNS" entries

Appedn the following lines to the end of your /etc/hosts file.

        100.64.0.32    seccomm.secdev.acep.uaf.edu
        100.64.0.32    auth.secdev.acep.uaf.edu
        100.64.0.32    cryptpad.secdev.acep.uaf.edu
        100.64.0.32    mattermost.secdev.acep.uaf.edu
        100.64.0.32    nextcloud.secdev.acep.uaf.edu
        100.64.0.32    owncloud.secdev.acep.uaf.edu
        100.64.0.32    plane.secdev.acep.uaf.edu

Linux and Mac

Open a terminal and run the following command:

sudo nano /etc/hosts

Windows

  1. Open Notepad as Administrator.
  2. Open the file: C:\Windows\System32\drivers\etc\hosts
  3. Edit and save.

3. Open SecComm websites

  • https://seccomm.secdev.acep.uaf.edu/
